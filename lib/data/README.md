# MITRE ATT&CK technique data

`mitre-enterprise-techniques.json` is generated from MITRE's official Enterprise
ATT&CK STIX 2.1 bundle. It includes active techniques and sub-techniques only;
revoked and deprecated objects are omitted.

Refresh the catalog with:

```sh
npm run update:mitre
```

Source: https://github.com/mitre-attack/attack-stix-data
